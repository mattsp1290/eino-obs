# Examples And Consumer Documentation

Use this file for documentation and example plans that show how `eino-agent`
will call into `eino-obs`.

## Pending Content

## Constraint

Examples should demonstrate adapter-friendly inputs without importing concrete
`eino-agent`, provider, or AG-UI runtime types unless a later adapter package is
explicitly designed.
